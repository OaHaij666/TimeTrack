# ⏰ TimeTrack 
一款专为手机设计的轻量级时间追踪应用，帮助你记录和管理日常任务时间，培养专注习惯。
# ✨ 主要功能
## 📝 任务管理
- **快速创建任务**：一键添加新任务，支持自定义名称和颜色标签
- **任务颜色标记**：8种可爱配色，让任务分类一目了然
- **重复任务**：支持设置每日、每周、工作日重复，省去重复创建的麻烦
## ⏱️ 计时功能
- **内联计时**：在任务卡片内直接开始/暂停计时，无需跳转页面
- **后台持续计时**：即使关闭浏览器或手机锁屏，计时依然准确运行
- **系统时间校准**：基于系统时间戳计算，确保时间记录精准无误
## 📅 日历视图
- **月度概览**：清晰展示整月任务分布
- **日期选择**：点击日期查看当天所有任务
- **快速添加**：在特定日期直接创建任务
## 📊 数据统计
- **今日/本周/本月统计**：实时查看不同时段的专注时长
- **历史对比**：与过去数据对比，了解自己的进步
- **任务排行**：查看哪些任务占据了你的时间
## 💾 数据存储
所有数据均保存在手机本地（LocalStorage），无需注册账号，保护隐私安全。
> ⚠️ 清除浏览器数据会导致记录丢失，重要数据请自行备份
